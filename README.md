# 🌳 Family Tree App

**Family Tree App** — вебдодаток для візуалізації родинних зв'язків, побудований на стеку **React + Firebase**. Проєкт дозволяє створювати інтерактивні генеалогічні дерева з автоматичним розташуванням вузлів та надійним хмарним збереженням.

## ✨ Основні можливості

### 🔐 Автентифікація та Профіль

- **Реєстрація:** Створення акаунту через Email/Password.
- **Безпека:** Захищені роути (доступ до дерева лише після авторизації).
- **Профіль:** Керування особистими даними користувача.

### 👨‍👩‍👧‍👦 Візуалізація Дерева

- **ReactFlow:** Інтерактивне полотно з підтримкою масштабувння та перетягування.
- **Smart Layout:** Автоматичне розташування вузлів (батьки завжди над дітьми).

### 👤 Картки осіб та Редагування

- **Життєпис:** Дати народження/смерті, біографія та цікаві факти.
- **Медіа:** Завантаження аватарів та створення фотогалерей.
- **Зв'язки:** Інтерфейс для призначення батьків та подружжя.
- **Валідація:** Розумні форми, що запобігають логічним помилкам.

## 🛠 Використані технології

- **Core:** React 18+ (Vite) + TypeScript
- **State:** Zustand
- **Backend:** Firebase (Firestore, Auth, Storage)
- **UI/UX:** Tailwind CSS, Framer Motion, Lucide Icons
- **Graph Engine:** ReactFlow + Dagre (для розрахунку координат)
- **Components:** Radix UI + Sonner

---

## 📂 Структура проєкту

- `src/store` — Глобальні сховища Zustand.
- `src/firebase.js` — Конфігурація сервісів Google.
- `src/components` — Кастомні вузли дерева та UI-компоненти.
- `src/pages` — Сторінки (Landing, Login, Tree, Profile).
- `public/_redirects` — Конфіг для SPA роутингу на Netlify/Vercel.

## 🏁 Початок роботи

### 🌐 Демо режим

Ви можете протестувати систему, використовуючи будь-яку пошту та пароль (мінімум 6 символів).

### 📍 Навігація (Routes)

- `/` — Головна сторінка.
- `/login` / `/register` — Вхід та реєстрація.
- `/tree` — Робоча область дерева (потрібна авторизація).
- `/profile` — Налаштування користувача.

## 📱 Інструкції користувача

> **➕ Як додати нову особу?**
>
> 1. На панелі керування натисніть **"Додати особу"**.
> 2. Заповніть обов'язкові поля (Ім'я, Прізвище, Стать).
> 3. Натисніть "Зберегти" — дерево перемалюється автоматично.

> **🔍 Як переглянути деталі?**
>
> - Клацніть на картку особи. Відкриється панель із табами: _Біографія_, _Фото_, _Сім'я_.
> - Для редагування натисніть іконку олівця у верхньому куті картки або панелі.

## 🚀 Заплановані фічі

Ці функції знаходяться у черзі на розробку:

- **🔄 Покращений Dagre:** Більш складні алгоритми для великих дерев (1000+ осіб).
- **🔗 Sharing:** Генерація посилання для перегляду дерева гостями.
- **👨‍👩‍👧‍👦 Мульті-дерева:** Можливість вести кілька незалежних родоводів.
- **📝 Activity Log:** Історія змін для спільного редагування.
- **📋 List View:** Табличний вигляд для швидкого пошуку та фільтрації.
